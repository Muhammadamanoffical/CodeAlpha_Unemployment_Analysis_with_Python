# 📊 Unemployment Analysis with Python (CodeAlpha Project)

## 🚀 Project Overview

This project analyzes unemployment trends using Python. It explores unemployment rate data to understand how employment levels changed over time, especially during the **COVID-19 pandemic**.

The project is part of the **CodeAlpha Data Science Internship**.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze unemployment trends over time
* Study the impact of COVID-19 on unemployment
* Visualize unemployment patterns across regions/states
* Extract insights for economic understanding

---

## 📂 Dataset Information

The dataset contains unemployment statistics including:

* 📍 State/Region
* 📅 Date
* 📊 Unemployment Rate (%)
* 👷 Employed Population
* 📉 Labour Participation Rate
* 🌍 Area (Urban/Rural)

---
